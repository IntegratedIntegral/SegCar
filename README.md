# SegCar

Arduino based robot that is supposed to keep balance (like a segway). Components used:</br>
-MPU 9250 accelerometer</br>
-2 DC motors (5V)</br>
-Arduino Uno</br>
-L293D motor controller</br>
-2 Lithium-ion batteries</br>
-2 wheels</br>
-Step-down converter</br></br>
